EAS Tools - Credits/Acknowledgements

Credits/Acknowledgements

Special thanks goes out to the following individuals, resources, and packages/organizations for their contributions to the development of EAS Tools:

Libraries & Packages

  • Hack font — Monospace font used for the site's UI typography.
  • wavefile.js — WAV file writer used for generating downloadable WAV audio files in the Encoder and Audio Splicer tools.
  • wave-resampler.js — Audio resampling library used for converting audio sample rates in the Encoder and Audio Splicer tools.
  • gif.js — GIF encoding library used to export text crawl animations as GIF files in the Text Crawl Generator.
  • piper.tts.js — WebAssembly-based TTS voice engine used for local in-browser speech synthesis in the Encoder and Audio Splicer tools.
  • nanotts.js — Additional WebAssembly TTS voice engine used alongside piper.tts.js for local in-browser speech synthesis.
  • CodeMirror — Code editor component used for input areas across multiple tools.
  • E2T-NG (JS port by wagwan-piffting-blud, originally by Newton Communications, see below for link) — Converts SAME headers into human-readable alert text; used in the Decoder and Text Crawl Generator.
  • sox-emscripten — SoX audio processing compiled to WebAssembly, used for audio manipulation in the Audio Splicer tool.
  • coi-serviceworker.js — Service worker that enables cross-origin isolation, required for SharedArrayBuffer support used by WebAssembly-based tools.
  • canvas-capture.js — Canvas recording library used to export text crawl animations as WebM video files in the Text Crawl Generator.
  • ffmpeg.wasm — FFmpeg compiled to WebAssembly, used for combining video and audio files in the Video/Audio Muxer tool.
  • phonemeize — Phoneme conversion library used for converting text into phoneme representations in the Phoneme Converter/Generator tool.
  • EAS.js (by GWES) — Reference implementation used for ENDEC profile logic in the Decoder and Encoder tools.
  • Original SAME decoding/encoding logic by CryptoDude3 (GitHub Pages site removed), maintained and expanded by wagwan-piffting-blud.

Inspiration & References

  • BunnyTub for the new EAS Tools "Toolbox" Logo
  • nicksmadscience SAME Encoder (Python) — Early reference for SAME header encoding logic.
  • Mab879 C++ SAME Encoder — Additional reference implementation for EAS/SAME encoding.
  • Anon64 EAS Header Generator (removed) — Additional reference for header generation.
  • EAS2Text (by Newton Communications) — Original Python EAS-to-text library that E2T-NG is based on.
  • CryptoDude3 GitHub Pages site (removed) — Source of the original encoder/decoder logic and the basis for the current page design.
